# Git e GitHub

# O que é Git?
Git é um sistema que serve para controlar versões do código. Tipo, ele guarda o histórico de tudo que foi feito no projeto.

# O que é GitHub?
GitHub é um site onde a gente salva os repositórios na internet. Dá pra acessar de qualquer lugar e também trabalhar em grupo.

# Comandos que usei

- git init -> cria o repositório
- git add -> adiciona os arquivos
- git commit -m "mensagem" -> salva uma versão
- git push -> envia pro GitHub

# Branch
Branch é tipo uma “cópia” do projeto pra testar coisas sem mexer no principal.
